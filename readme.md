## Getting Started (Only Tested on Lunix so far)
To run this program you will need ollama installed on your machine.
To get started with the project, follow these steps:

1. Clone the repository to your local machine using `git clone https://github.com/yourusername/llamaTest.git`
2. Navigate to the project directory using `cd llamaTest`
3. Install any necessary dependencies by running `chmod +x run.sh` 
4. Run the project using the appropriate command, such as `./run.sh`
5. Program should be accessiable using `llama` command in lunix after setup.
